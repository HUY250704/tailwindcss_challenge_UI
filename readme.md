# Login & Register UI

# Objective
Thiết kế 2 trang:
- Login
- Register  

Sử dụng **HTML + TailwindCSS** để implement UI từ Figma.

---

# Requirements
Bài làm cần đảm bảo:

- Hiểu và áp dụng **utility-first approach**
- Viết **HTML structure rõ ràng, clean**
- Sử dụng đúng **layout system (Flexbox / Grid)**
- Có **responsive design**
- Code **dễ đọc, dễ maintain**

---

# Constraints

# Được phép sử dụng
- HTML  
- TailwindCSS (utility classes)  

## Không được sử dụng
- Custom CSS  
- Inline style  
- CSS framework khác  
- Component library
